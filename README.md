# 💰 ردیاب قیمت نقره - Streamlit

![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)

اپلیکیشن تحت وب برای ردیابی قیمت لحظه‌ای نقره (جهانی و ایران) با رابط کاربری فارسی.

## ✨ ویژگی‌ها

- 🌍 **قیمت جهانی نقره** (دلار/اونس)
- 🇮🇷 **قیمت نقره در ایران** (تومان/گرم)
- 📊 **نمودارهای تعاملی** با Plotly
- ⚡ **به‌روزرسانی لحظه‌ای**
- 📱 **طراحی واکنش‌گرا** برای موبایل و دسکتاپ
- 🔢 **محاسبه آمار و پریمیوم بازار**
- 🎨 **رابط کاربری فارسی و مدرن**
